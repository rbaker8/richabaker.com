I have been a software developer working at software companies for almost my entire 25 year career, in banking, public safety, cybersecurity and public education. This site contains my professional portfolio, github projects I have developed to refine and showcase my technical skills.

The first project, Employees, is a Spring Boot and Spring 5 REST microservice running on an embedded Tomcat server with an in-memory H2 SQL database backend for demonstration purposes.

The second project, Collections, is a Java class library of my implementations of many of the classic data structures such as lists, deques, queues, stacks, maps, sets and trees. This includes Test-Driven-Development unit tests of these collection implementations.
